\chapter[caracteres speciaux]{Caractères spéciaux}

\index{caractères spéciaux}

Vous avez remarqué que les commandes \CONTEXT\ étaient précédées par
un \tex{} (antislash). \tex{} à donc une signification spéciale pour
\CONTEXT. À part \tex{}, il existe d'autres caractères qui nécessite
une attention particulière si vous souhaitez qu'ils apparaissent en
mode verbatim ou en mode texte. Le \in{tableau}[tab:specchars]
présente une vue d'ensemble de ces caractères ainsi que ce que vous
aurez à taper pour les utiliser dans vos documents.

\let\normalunderscore=\_
\let\normaltilde     =\~

D'autres caractères spéciaux possèdent une signification spéciale lors
de la composition d'expressions mathématiques et certains d'entre eux
ne peuvent être utilisés que dans le mode math
(cf. \in{chapitre}[formulas]).
